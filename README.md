# Mining-NBA-Data-and-Cluster-Players-  
Crawled and Preprocessed various data of NBA teams and players over past 30 years using nba_py API

Analysis back-to-back game win-rate of teams and players and visualize the results using Spark(scala)

Used T-sne to decrease the dimension of the players’ data and applied Affinity Propagation Clustering (python) to players’ data of each year to find out the change of game style in NBA over past 30 years
